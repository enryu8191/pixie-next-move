# PixieNextMove Project Specification

**AI-Powered Next-Move Analyzer for Pixie Chess**

**Date:** May 2026
**Target:** CODEX development team

## Overview
PixieNextMove is a web application modeled after chessnextmove.com but specialized for Pixie Chess (pixiechess.xyz).

Pixie Chess features collectible low-poly 3D pieces with unique magical abilities. Players build their army from their collection. Standard chess rules apply except when overridden by piece abilities.
